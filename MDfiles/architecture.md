## 🏗️ Описание ключевых компонентов

### 1. Ядро логики (`includes/functions.php` и `config/database.php`)
*   **Инициализация:** В `database.php` происходит загрузка переменных окружения (из Docker), создание PDO-соединения с БД, инициализация параметров приложения и запуск безопасной PHP-сессии.
*   **Data Access Layer:** В `functions.php` находятся функции для прямого обращения к БД с использованием **Prepared Statements** (например, `getProducts()`, `getProductById()`).
*   **Бизнес-логика:** Корзина хранится в пользовательской сессии (`$_SESSION['cart']`), функции для её перерасчета и применения промокодов находятся в `functions.php`.
*   **Безопасность:** Реализованы функции для генерации и проверки CSRF-токенов (`generateCsrfToken()`, `validateCsrfToken()`), лимитирования запросов (`checkRateLimit()`) и аутентификации. Мультиязычность реализована через `lang.php`.

### 2. Точки входа (PAGES)
Страницы (`index.php`, `catalog.php`, `cart.php`, `login.php` и др.) исполняют роль как **View** (отрисовка HTML-разметки), так и легковесных **Controllers** (получение данных из `functions.php` и их передача в шаблон). За счет `includes/header.php` и `includes/footer.php` достигается переиспользование общих UI элементов.

### 3. API Эндпоинты (`api/`)
Принимают асинхронные запросы (AJAX/Fetch) от клиентского JavaScript-кода (например, из `main.js`), обрабатывают их (проверка CSRF, вызов нужных функций из `functions.php`) и возвращают ответы строго в формате **JSON**. Это позволяет реализовать динамическое обновление корзины или применение промокода без перезагрузки страницы.

### 4. Инфраструктура (Docker)
Проект полностью контейнеризован при помощи Docker:
*   Веб-контейнер (`web`) собирается на основе `Dockerfile` из официального PHP-изображения с Apache (mod_rewrite включен).
*   Контейнер базы данных (`db`) использует MySQL 8.
*   Утилита администрирования (`phpmyadmin`) доступна «из коробки» для удобного управления БД.
Все сервисы связаны внутренними сетями Docker и конфигурируются переменными окружения (`.env`).
